==============

INSTALL
~~~~~~~~~~~~~~~
install from source:

::

    git clone https://github.com/liujinliu/anan.git
    cd anan
    make install
    make uninstall

USEAGE
~~~~~~~~~~~~~
::

	$cruiser-run --config_file filename
